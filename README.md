## Philosophy

> "The operating system should disappear so the user can focus only on what matters."

PED OS is built around four principles: **elegant**, **fast**, **silent**, **focused**.

## Stack

| Layer | Technology |
|---|---|
| Kernel | Linux |
| Display Server | Wayland |
| Compositor | Hyprland (fork) |
| Rendering | Vulkan / OpenGL |
| Core | Rust, C++ |
| Interface | Qt / QML |

---

## Components

| Package | Description |
|---|---|
| `ped-shell` | Main desktop interface |
| `ped-dock` | Minimalist dock |
| `ped-launcher` | Universal search & app launcher |
| `ped-settings` | Control panel |
| `ped-store` | App store |
| `ped-files` | File manager |
